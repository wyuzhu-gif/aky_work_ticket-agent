# 受限空间安全措施

> 受限空间票审查"风险控制措施"和"现场条件"维度的核心依据。关联：[[gb30871-2022-confined-space]] | [[gb30871-2022-general]] | [[气体检测要求]] | [[受限空间作业审批要求]]

## 一、安全隔离（6.1）

1. **管道隔离** — 采用加盲板或拆除一段管道方式，不应采用水封或关闭阀门代替盲板
2. **孔洞封堵** — 与受限空间连通的可能危及安全的孔洞应严密封堵
3. **电源断电** — 电器电源应可靠断电，电源开关处上锁并加挂警示牌

## 二、通风要求（6.2）

4. **自然通风优先** — 打开人孔、手孔、料孔、风门等与大气相通的设施
5. **强制通风** — 必要时采用强制通风或管道送风，送风前分析确认管道内介质和风源
6. **忌氧环境** — 通风前应对与氧性质相抵的物料采取卸放、置换或清洗合格的措施

## 三、个体防护装备（6.6）

| 受限空间类型 | 必须防护装备 |
|-------------|------------|
| 缺氧或有毒（清洗置换仍不达标） | 隔绝式呼吸防护装备 + 救生绳 |
| 易燃易爆（清洗置换仍不达标） | 防静电工作服/鞋 + 防爆工器具 |
| 酸碱腐蚀性 | 防酸碱防护服/鞋/手套 |
| 电焊作业 | 绝缘鞋 |
| 噪声 | 耳塞/耳罩 |
| 粉尘 | 防尘口罩（满足GB 15577和GB 39800.1） |
| 高温 | 高温防护用品 + 通风隔热 |
| 低温 | 低温防护用品 + 供暖 |
| 清污作业 | 隔绝式呼吸防护装备 + 救生绳 |

7. **通信工具** — 受限空间内作业应配备通信工具（6.6j）

## 四、监护人特殊要求（6.8）

8. **外部全程监护** — 不应在无防护措施下探入或进入受限空间
9. **增设监护** — 风险较大时应增设监护人员，随时保持联络
10. **出入登记** — 对进入人员及携带工器具种类、数量进行登记，作业完毕后清点

## 五、其他安全要求（6.7, 6.9）

11. **交叉作业禁止** — 存在动火作业时，不应安排涂刷油漆/涂料等可能产生有毒有害、可燃物质的作业
12. **出入口畅通** — 受限空间出入口应保持畅通
13. **禁止携带无关物品** — 不应抛掷材料、工器具
14. **轮换作业** — 难度大、劳动强度大、时间长、高温的应采取轮换作业
15. **异常处置** — 未穿戴规定防护装备的人员严禁入内救援
16. **停工标识** — 停止作业期间入口处增设警示标志，防止误入

## 六、票证时效

- 受限空间安全作业票有效期不超过24h（6.10）

## 审查交叉核验要点

- 安全隔离措施是否包含盲板隔离（而非仅关阀门）
- 个体防护装备是否与受限空间类型匹配
- 监护人是否在外部全程监护（不能在空间内监护）
- 出入人员及工器具是否登记清点
